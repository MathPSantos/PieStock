## Development
> **yarn** is required

After cloning this repo
### Mobile
#### Android
 - Start **Android** Emulator
 - Open the **mobile** folder
 - Run ```yarn android``` and have fun!

#### IOS
 - Start **IOS** Emulator
 - Open the **mobile** folder
 - Run ```npx pod-install ios```
 - Run ```yarn ios``` and have fun!
